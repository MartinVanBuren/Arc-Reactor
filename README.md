# Arc-Reactor
The arduino code for my LED arc reactor.
### Dependencies
- [Adafruit_NeoPixel library](https://github.com/adafruit/Adafruit_NeoPixel)
- [Digispark Pro Drivers](http://digistump.com/wiki/digispark/tutorials/connectingpro)
